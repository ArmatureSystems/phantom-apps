Asset Configuration

By using the cache_limit configuration parameter you can manage how old(in days) cached images do you accept. Allowed values are 0-14. A zero value means always download fresh screenshots. If you need shorter interval than one day, you can use decimal numbers as parameter, e.g. cacheLimit=0.041666 means: use image from cache only if it is not older than 1 HOUR (1/24=0.041666). In case of a value outside of the allowed range, default value 0 will be used.
